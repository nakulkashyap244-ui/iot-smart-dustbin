# IoT Smart Dustbin – Python + Flask REST API

Mini project simulating an IoT-based smart dustbin.

## Features

- Python script simulates a dustbin sensor sending garbage level values (25%, 70%, 90%, 100%).
- Flask REST API receives the sensor data and returns real-time dustbin status as JSON.
- Triggers a warning alert when dustbin reaches 100% capacity.
- Demonstrates IoT workflow: Sensor Script → API → JSON Response → Alert.

## Tech Stack

- Python
- Flask (REST API)
- JSON
